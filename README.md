# Battleship - The Odin Project

The good old Battleship game you played as a kid (sort of)

- [Live demo](https://adrianogtl.github.io/top-battleship)
- [Project assignment](https://www.theodinproject.com/lessons/node-path-javascript-battleship#assignment)

## Features

- Place your ships horizontally or vertically on the board
- Play against the computer
